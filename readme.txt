== Description ==

Plugin to extend features of Business Hub Pro Theme. This plugin helps to add event, pricing table and testimonial and also shows twitter and instagram feeds for the Business Hub Pro theme.

== Features include ==

* Show upcoming events
* Display events(pasts, future)
* Design options available to choose to display event list 
* Shortcodes in tinymce and widgets available
* Unlimited features for pricing table
* Show twitter and instagram feed
* Fully Responsive- compatible with mobile devices

== Installation ==

= Using The WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Search for `business-hub-toolbox`
3. Click `Install Now`
4. Activate the plugin on the Plugin dashboard

= Using FTP =

1. Download  Business hub toolbox from WordPress.
2. Unzip business-hub-toolbox.zip
3. Upload all the files to the /wp-content/plugins/business-hub-toolbox
4. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==

= Do this plugin work for other themes ? =
This plugin adds custom post types, meta fields and widgets. These elements can be used in other themes as well.

= Does twitter widget works on localhost ? =
No, Twitter widget works on live site only. It doesn't work on localhost currently.
